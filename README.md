# Public transport in Poznan
The application displays selected public transport stops in the city.

### Tech used
* HTML
* CSS
* Bootstrap
* JavaScript
* Leaflet

### Sources 
* Map's layer:
https://openstreetmap.org/
* Stops informations:
http://www.poznan.pl/api/
